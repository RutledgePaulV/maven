### Maven
Parent poms for com.github.rutledgepaulv projects. Includes basic configuration
for gitflow, tests, code coverage, and releasing.


### Usage
```xml
<parent>
    <groupId>com.github.rutledgepaulv</groupId>
    <artifactId>maven</artifactId>
    <version>1.0</version>
</parent>
```
